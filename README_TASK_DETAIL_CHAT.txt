Neue Aufgaben-Funktionen:

- Aufgabenliste zeigt keine Beschreibung mehr.
- Jede Aufgabe kann über „Öffnen“ auf einer Detailseite geöffnet werden.
- Detailseite enthält Beschreibung und gespeicherten Aufgaben-Chat.
- Chat speichert Nutzername, Nachricht und Zeitstempel in tasks.json.
- Aufgaben können mehrere bekannte Zuständige haben.
- Zusätzlich können manuelle Zuständige per Komma eingetragen werden.
- Für jeden bekannten Zuständigen wird beim Erstellen eine DM/Channel-Benachrichtigung in die bestehende Notification-Queue geschrieben.

Nach dem Hochladen Browser hart neu laden: STRG + F5.
